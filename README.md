## About Me

I'm a DevOps Leader and Cloud Architect based in Dubai 🇦🇪, with 13+ years designing scalable, secure, and automated infrastructure. Currently leading the DevOps function at **Almosafer**, where I own cloud architecture, platform engineering, observability, and release reliability for a large-scale travel platform.

Most of what I build day-to-day is proprietary enterprise infrastructure, so this profile leans on public side projects and the tech I work in — for real case studies and career history, [rolind.me](https://rolind.me) has the full picture.

## 🤖 What I'm Building Right Now

Infrastructure knowledge tends to get locked inside dashboards, runbooks, and the heads of senior engineers — so I've been building internal tooling that closes that gap with AI:

- Natural language interfaces over infrastructure state
- Intelligent log analysis and anomaly surfacing
- Context-aware alerting that answers "so what," not just "what changed"

The goal isn't AI for its own sake — it's reducing operational toil for the engineer paged at 2am.
